# HTML Is All Agents Need — James Russo, HeyGen

[![HTML Is All Agents Need — James Russo, HeyGen](https://i.ytimg.com/vi/Cz4v1WHVyZc/hqdefault.jpg)](https://www.youtube.com/watch?v=Cz4v1WHVyZc)

## TL;DR
James Russo, co-creator of Hyperframes at HeyGen, argues that HTML, CSS, and JavaScript are the ideal native languages for AI agents to generate videos. By leveraging a thin HTML wrapper and a deterministic browser rendering technique called "freezing the clock," Hyperframes allows any coding agent to create high-quality, editable videos, bridging the gap between building a product and launching it with a polished video.

## Key Takeaways
- While coding agents have democratized software building, creating the launch videos to promote those products remains difficult.
- HTML, CSS, and JavaScript are the "native languages" of LLMs because most of their training data consists of scraped web pages; forcing them to use custom JSON or XML formats yields suboptimal results.
- HeyGen built Hyperframes, an open-source framework that turns an agent's HTML code into deterministic MP4 videos.
- The framework uses a "thin wrapper" of standard HTML with a few data attributes for timing metadata, which proved more effective than complex, heavily prompted wrappers.
- To solve the asynchronous nature of web browsers (where fonts and images load at different times), Hyperframes "freezes the clock" and seeks frame-by-frame, waiting for assets to load before taking a screenshot to build the video.
- Because rendering happens in the browser, Hyperframes can utilize any web technology for video, including Three.js, charts, SVGs, shaders, WebGL, WebGPU, and Lottie.
- Instead of teaching agents how to code, Hyperframes' "skills" focus on teaching agents taste, motion, and video design principles.
- Achieving great video output requires iterative craft—storyboarding, frame-by-frame motion design, and human-in-the-loop editing via the Hyperframes studio.
- Hyperframes has gained significant traction, rendering 1.3 million videos in 90 days and accumulating 32,000 GitHub stars.
- The creators are developing a "code to video benchmark" to help LLM labs improve the creative capabilities of future models, which currently still struggle with creative work.

## Detailed Breakdown

### The Challenge of Launching Products [00:36](https://www.youtube.com/watch?v=Cz4v1WHVyZc&t=36s)
Russo opens by showing a video entirely generated by an agent using HTML. He notes that while coding agents have made building software incredibly easy, creating the launch posts and videos to promote those products is still hard. HeyGen's mission is to solve communication through video, starting with AI avatars (A-roll), but recognizing that great videos also require B-roll, animations, captions, and music.

### HTML as the Native Language of LLMs [02:09](https://www.youtube.com/watch?v=Cz4v1WHVyZc&t=129s)
Russo explains HeyGen's bet on HTML, CSS, and JavaScript. Because LLMs are trained on scraped web pages, these web languages are their "native tongue." Forcing an LLM to learn a custom DSL or JSON format is likened to asking Shakespeare to write a poem in Japanese. Russo notes that others in the AI space, including Tarik and Andrej Karpathy, have independently arrived at the same conclusion: HTML is the new markdown for visual LLM output.

### Evaluating Frameworks and Choosing HTML [03:43](https://www.youtube.com/watch?v=Cz4v1WHVyZc&t=223s)
The team evaluated existing video creation tools on an axis of "quality of results" versus "agent-friendly." Professional tools like After Effects and Premiere Pro have high quality but act more as co-pilots and aren't agent-friendly. Frameworks like Lottie and Rive use custom formats that aren't native to LLMs and lack controllability. Remotion was promising but required teaching the agent the framework, which stifled creativity. Around November, with the release of newer Gemini models, the team saw a step-function improvement when models naturally gravitated toward HTML to generate desired outputs.

### Building Hyperframes with a Thin Wrapper [05:18](https://www.youtube.com/watch?v=Cz4v1WHVyZc&t=318s)
HeyGen started with Gemini 3 Flash as a "design partner," reasoning that if a small model could author workable code, larger models could too. They tried adding complex context and system prompts, but the "thinnest wrapper"—essentially plain HTML with a few data attributes for timing metadata—ultimately won. The preview and render occur entirely in the browser, meaning whatever the browser sees is exactly what the video will be.

### Deterministic Video Rendering [07:21](https://www.youtube.com/watch?v=Cz4v1WHVyZc&t=441s)
A major challenge was converting asynchronous browser rendering into deterministic MP4 video. Browsers load elements like fonts and images at different times, which is unacceptable for video. Hyperframes solves this by "freezing the clock" and seeking frame-by-frame. The system waits for everything on the page to load, takes a screenshot, and moves to the next frame, repeating the process until all frames are captured and encoded into a video.

### Browser Rendering Capabilities [08:56](https://www.youtube.com/watch?v=Cz4v1WHVyZc&t=536s)
Because Hyperframes renders in the browser, any web-renderable technology can be used in a video. This includes Three.js, charts, SVGs, shaders, WebGL, WebGPU, and Lottie. Russo notes that a big part of their workflow involves finding inspiration on the internet, tweaking web examples, and putting them into videos.

### Teaching Agents Taste and Video Skills [09:28](https://www.youtube.com/watch?v=Cz4v1WHVyZc&t=568s)
Since LLMs already know how to write HTML, Hyperframes' "skills" focus on teaching agents video design principles, taste, and motion. This allows them to raise the floor of single-shot prompts. For example, a "website to video" skill can scrape a site for branding and assets, apply basic motion principles, and generate a decent video automatically.

### The Craft of Great Video Output [11:01](https://www.youtube.com/watch?v=Cz4v1WHVyZc&t=661s)
Russo draws a parallel to AI coding: single prompts yield decent results, but great output requires craft. Power users are encouraged to storyboard frame-by-frame, add motion iteratively, and use the Hyperframes studio for last-mile editing. HeyGen also just released keyframes in Hyperframes, allowing users to coordinate motion like a professional motion designer in After Effects.

### Adoption and Open-Source Success [12:38](https://www.youtube.com/watch?v=Cz4v1WHVyZc&t=758s)
Hyperframes is open-source and free forever. It has rendered 1.3 million videos in 90 days, with 267,000 creators trying it and 15,000 videos rendered daily. It has 32,000 GitHub stars. It works with any coding agent (Claude Code, Codex, Cursor) because if an agent can write HTML, it can make a Hyperframes video.

### The Future: Code to Video Benchmark [13:40](https://www.youtube.com/watch?v=Cz4v1WHVyZc&t=820s)
Russo honestly admits that models are still not great at creative work. To push the industry forward, HeyGen is working on a "code to video benchmark" with LLM labs and creators to raise the floor of AI video generation. He invites interested collaborators to reach out.

### AI Engineer Warfare Showcase [14:10](https://www.youtube.com/watch?v=Cz4v1WHVyZc&t=850s)
As a closing surprise, Russo shows a collaboration with the AI Engineer team on a Warfare showcase video. He reiterates the core thesis: agents have made building easy, launching is still hard, and HTML is all agents need to make great launch videos.
