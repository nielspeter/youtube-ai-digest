## TL;DR
Anthropic's Lamis discusses the newly released Claude Tag, a proactive Slack-based agentic teammate that orchestrates end-to-end workflows—from customer feedback to code implementation and PR creation—without requiring developers to leave their chat environment. Internally, Claude Tag has already driven massive adoption, opening 65% of product engineering PRs, while Anthropic's broader dogfooding culture and innovations like "dreaming" (automated memory optimization) showcase the future of asynchronous, multi-player agentic development.

## Key Takeaways
- **Claude Tag is a proactive teammate in Slack:** Unlike simple Slack bots, Claude Tag operates autonomously over long periods, can initiate conversations, and possesses cross-channel memory and context.
- **Massive internal adoption at Anthropic:** 65% of product engineering PRs at Anthropic are currently opened by Claude Tag, making it the team's daily driver.
- **Shift from single-player to multi-player workflows:** Claude Tag enables cross-functional collaboration where engineers, product managers, and sales folks can all be looped into a single, transparent agentic workflow.
- **Asynchronous, long-running tasks:** Unlike Claude Code which is highly interactive and session-based, Claude Tag leans into running asynchronous tasks for hours and returning with completed features.
- **Agent identities and permissions:** Claude Tag operates with its own permissions and API keys rather than assuming an individual user's credentials, making it easier to audit and scale safely across enterprises.
- **The METR chart trend:** The time horizon for which agents can autonomously complete tasks is roughly doubling every four months, driving increased trust in AI.
- **Simplification over time:** As models improve, Anthropic actively removes abstractions and shrinks the harness, trusting the model to handle capabilities like memory management via simple file systems.
- **"Dreaming" for continual learning:** Anthropic introduced a research preview feature called "dreaming" that reviews agent session transcripts and memory stores to identify and fix discrepancies, enabling automated continual learning.
- **Company-wide empowerment:** Non-technical teams at Anthropic, including marketing and sales, are heavily using Claude Code and Claude Tag to automate workflows, generate reports, and triage incidents.
