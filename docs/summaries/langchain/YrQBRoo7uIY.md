## TL;DR
LangSmith Sandboxes provide every AI agent with its own secure, isolated computer environment so agents can safely write code, run tests, and perform real work without risking the host system. Sandboxes spin up in under a second, support package installation and file editing, and can scale from a single instance to thousands in production.

## Key Takeaways
- Agents have evolved from answering questions to writing code, running tests, and analyzing data—tasks that require a full computer environment, not just models and tools.
- Giving an agent direct access to a laptop or host machine is unsafe because agents run untrusted code by default, risking unexpected consequences.
- LangSmith Sandboxes give each agent its own isolated, secure computer environment.
- Sandboxes spin up in under a second, enabling rapid provisioning.
- Supported operations include installing packages, running scripts, editing files, and pausing/resuming across long sessions.
- Users can control access, snapshot, and fork sandbox environments.
- The system scales seamlessly from a single sandbox for one agent to thousands for production deployments.
- The core challenge is not whether agents need computers, but how to provide them safely.
