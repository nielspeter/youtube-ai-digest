## TL;DR
OpenAI introduces "ChatGPT Work," a new evolution of ChatGPT across web, desktop, and mobile that transforms the assistant from a question-answering tool into an active work delegate. By connecting to external apps, local files, and even directly controlling the user's computer and browser, ChatGPT can now autonomously handle multi-step workflows across marketing, data analytics, and engineering.

## Key Takeaways
- ChatGPT shifts from merely answering questions to actively doing work by connecting to your tools and local files.
- Users can delegate real tasks—like daily briefings or campaign creation—and set up recurring automations without needing to ask repeatedly.
- Plugins connect ChatGPT to everyday apps (Slack, Drive, Calendar) and bundle role-specific workflows for fields like sales, data science, and marketing.
- The experience is seamless across platforms: start a chat on web and continue it on mobile.
- The desktop app integrates local files and projects, giving ChatGPT full context to work autonomously.
- ChatGPT can generate brand-consistent marketing visuals and mood boards from a logo and brand guidelines.
- For data analytics, ChatGPT can unify metrics from multiple sources, flag anomalies, and build live, shareable dashboards.
- "Sites" lets users publish websites and dashboards directly from ChatGPT to share with teams.
- Codex is now integrated inside ChatGPT, allowing users to review code diffs and publish pull requests.
- "Appshots" and background Computer Use let ChatGPT directly interact with apps and browsers when no dedicated plugin exists.
