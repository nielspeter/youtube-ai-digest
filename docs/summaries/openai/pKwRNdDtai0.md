## TL;DR
OpenAI demonstrates how ChatGPT's new plugins directory lets you securely connect external apps—like Gmail, Slack, Google Drive, and Canva—directly to ChatGPT. Plugins bundle "skills" that give ChatGPT a playbook for real work tasks, from designing event flyers to preparing for customer meetings, all within a single conversational interface.

## Key Takeaways
- ChatGPT now features a plugins directory where users can connect tools they already use, such as Gmail, Slack, and Google Drive.
- Plugins let you securely link an app to ChatGPT, enabling it to access and pull information from those connected services.
- Each plugin includes "skills" that act as playbooks for specific jobs—structuring a report, preparing for a meeting, or guiding a repeatable workflow.
- Creative tasks are covered too: you can ask ChatGPT to build design assets (e.g., an event flyer) using a tool like Canva, then jump into Canva to make edits.

- ChatGPT can save its output (e.g., a meeting brief) as a new document and share it with the team.
- The underlying pattern—gathering context, doing work, and producing an output—can be applied to any type of task inside ChatGPT.
