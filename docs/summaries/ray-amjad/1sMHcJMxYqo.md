# Claude Code + Anki = Learn ANYTHING!

[![Claude Code + Anki = Learn ANYTHING!](https://i.ytimg.com/vi/1sMHcJMxYqo/hqdefault.jpg)](https://www.youtube.com/watch?v=1sMHcJMxYqo)

## TL;DR
Ray Amjad demonstrates how to combine Claude Code (an AI agent that runs on your computer) with Anki (a spaced repetition flashcard app) to accelerate learning. By installing the Anki Connect add-on, Claude Code can automatically create, manage, and optimize flashcards from videos and lectures, identify knowledge gaps, and close the entire learning loop by cross-referencing your flashcards with your real-world performance.

## Key Takeaways
- **Anki Connect Add-on**: The bridge between Claude Code and Anki is the Anki Connect add-on, which allows external programs to interact with Anki's database.
- **Automated Flashcard Creation**: Claude Code can generate Anki flashcards from YouTube videos, local lectures, and textbooks, complete with diagrams, screenshots, and timestamps.
- **Focus on Understanding**: This workflow lets you focus 100% of your attention on understanding a lecture or video while watching it, leaving the active recall and flashcard creation to Claude Code.
- **Leech Surgery**: Claude Code can identify "leeches" (flashcards you repeatedly fail), diagnose why they are difficult, and rewrite them to be easier to understand.
- **Laddering**: When you struggle with a concept, Claude Code can automatically create intermediate flashcards to bridge the knowledge gap.
- **Confusable Pairs & Diversification**: The AI can detect cards you commonly mix up and create distinction cards, or add varied examples so your recall generalizes beyond a single context.
- **Closing the Learning Loop**: By connecting Claude Code to meeting recording tools (like Granola via MCP), you can evaluate whether you are actually applying the concepts you learn in Anki to your day-to-day work.
- **Refactoring Cards**: Claude Code can break down massive walls of text into atomic, minimum-information-principle-compliant flashcards.
